Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • Adds to the structure of HTML to add design
  • Makes pages pretty

Git

  • Version Control
  • git status: checks what branch you're currently
  • git checkout -b branch-name: Creates a new branch and switches to it
  • git pull: adds class repo to gitlab within the DU-class folder

JavaScript

  • Adds to HTML & CSS to create functionality
  • Creates an interactive User Experience

Extra Notes

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Short Cut Keys

  • Shift+Alt+F: reformats cleanly
  • Alt+L,Alt+O: Opens .html in Live Server
  • Alt+Up/Down Arrows: When line(s) highlighted, can move about
